# 📊 Goods Sales Dashboard

An AI-powered interactive sales analytics dashboard built using **Streamlit**, **Python**, **Pandas**, **Plotly**, and **Google Gemini AI**. The dashboard helps businesses visualize sales data, identify trends, generate forecasts, and receive AI-powered business insights.

---

## 🚀 Features

- 📈 Interactive Sales Dashboard
- 📊 Revenue, Profit & Sales Analysis
- 🌍 Region-wise Sales Visualization
- 📦 Product Performance Analysis
- 📅 Time-Series Sales Trends
- 🔍 Advanced Data Filtering
- 🤖 AI-Powered Insights using Google Gemini
- 📉 Sales Forecasting
- 📥 Export Filtered Data
- 📱 Responsive and User-Friendly Interface

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/manvadityasingh16/goods-sales-dashboard.git
```

### 2. Navigate to the project

```bash
cd goods-sales-dashboard
```

### 3. Create a virtual environment

**Windows**

```bash
python -m venv .venv
.venv\Scripts\activate
```

**Mac/Linux**

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Configure Google Gemini API

Create a `.env` file in the project root.

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

### 6. Run the application

```bash
streamlit run dashboard.py
```
